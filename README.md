# Ours Ledger

A couples-first budgeting platform for households that treat all money as shared.

## Stack

- Next.js 16 (App Router, TypeScript)
- Tailwind CSS 4
- Bun package manager

## Product Principle

"What's mine is ours." The default mode is complete financial visibility and shared decision-making.

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
bun run dev
bun run lint
bun run build
bun run start
```
